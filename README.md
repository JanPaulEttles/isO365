# isO365

## Project Set up
* git clone https://github.com/janpaulettles/isO365.git
* cd isO365
* npm init --yes
* npm install fast-csv
* npm install yargs
* npm install async
* npm install reuqest

## Add stuff
* git add .
* git commit -m "initial"
* git commit -a
* git push origin master

## Pull it (sorry devo)

* git pull
